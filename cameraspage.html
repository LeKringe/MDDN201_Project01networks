CAMERAS

Cameras are used for a variety of purposes, be it for security reasons, or to keep surveillance in an area. For a system of cameras to be of proper use, an effective network linking each camera needs to be established in order for easy retrieval of footage at any point in time at any location.

IP CAMERA

An IP camera, or Internet Protocol camera, is a type of security camera that receives control data and sends image data via the internet, linking it into a network of other security cameras.

SECURITY ALARM

Security alarms are alarms that go off when an intruder is detected. It shares the same network as the security cameras, and shares infromation with it, allowing it to be activated when a camera detects something.

4K SECURITY CAMERAS

Similar to normal security cameras, but are able to record footage in high resolution, picking up important details such as faces and license plate numbers.
